# Slack App

### 1. Go to [your Slack apps](https://api.slack.com/apps).

![](../../.gitbook/assets/image%20%2839%29.png)

### 2. Create New App.

Name the app `Toast`  
Select the workspace you are setting this up for.

![](../../.gitbook/assets/image%20%2832%29.png)
